NBDpsy 助手

使用指南

把这台电脑 Chrome 里登录的小红书账号,安全同步到 NBDpsy 运营后台。

1首次配置(只做一次)

  1. 打开插件弹窗,在「Operator API Key」里粘贴接入包里发给你的那串 apikey,点保存。
  2. 保存后底部出现「服务连接正常」即配置完成。
  3. 服务器地址一般不用动;需要改时展开「高级设置」。
apikey 忘了或失效:找管理员重置后重新下发,自己是查不回明文的。

2登录小红书账号(加号 / 换号)

  1. 先到 chrome://extensions → 本插件「详情」→ 勾选「在无痕模式下启用」(只需勾一次)。
  2. 回到插件弹窗,点「打开隐私窗口登录(加/换号)」。
  3. 在弹出的无痕窗口里用手机小红书 App 扫码登录。
  4. 登录成功后插件会自动把账号同步到后台,弹窗「我的账号」列表里会出现这个号。
每个号用独立的无痕窗口登录,互不串号;登录完成即可关掉无痕窗口。

3看懂「我的账号」

  • 有效账号登录态正常,后台可正常发布/拉数据。
  • 失效登录态过期——重新走一遍第 2 步扫码即可恢复。
  • 验证码被小红书风控拦下,需要人工打开该号过一次验证。
  • 点每个号右侧的「检测」可立即验活;点外链图标可直接打开该号主页。

4常见问题

列表里出现 xhs_account_ 一串数字的怪账号?
登录同步中断留下的占位号,重新扫码登录同一账号会自动清掉,系统也会定期回收。
点「打开隐私窗口登录」没反应?
八成是没勾「在无痕模式下启用」(见第 2 步第 1 条)。
怎么确认插件是不是最新版?
看 chrome://extensions 里本插件显示的版本号,与管理员通知的最新版本比对。弹窗右上角也有当前版本。
发布 / 拉数据在哪里操作?
不在插件里。插件只负责登录与账号同步;发布、笔记数据、删除笔记都由运营助手(Claude)通过后台完成,直接跟它说人话即可。
